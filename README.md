# Component Coupling Validator

This project was generated with [Angular CLI](https://github.com/angular/angular-cli) version 9.0.3. And is powered by a firebase backend.

## Development server

Run `ng serve` for a dev server. Navigate to `http://localhost:4200/`. The app will automatically reload if you change any of the source files.

## Firebase

To run this project by yourselve, you need to create a firebase project and link it to this project (i.e. `firebase init`). Furthermore, you will need to change the environment files under `src/environments/` to match your firebase config.
